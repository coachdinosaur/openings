# Catalan Atelier app

This is the local web application used by the root `start-local.ps1` launcher. It contains the complete guided Chapter 1 lesson from printed pages 8-23, legal interactive section entries, the manual review overlay, and the deterministic fixed-scope B2 PDF re-import.

For development with Node.js 22 or newer:

```powershell
npm install
npm run dev
```

Production verification:

```powershell
npm test
```
